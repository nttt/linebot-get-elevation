# linebot-get-elevation
LINEボットを勉強用に作成。<br/>
住所を入力すると、その住所を検索し、標高を返します。<br/>
現在地情報を送った場合は、その住所の標高を返します。<br/>

https://www.geocoding.jp<br/>
https://maps.gsi.go.jp/development/api.html<br/>
